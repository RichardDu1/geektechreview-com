## Step 1: Gather Equipment and Software

- **Measurement Microphone**: Use a calibrated USB mic like the **MiniDSP UMIK-1** (or similar).
- **Software**: Download and install **Room EQ Wizard (REW)** (free) on a laptop.
- **DSP**: Have a **MiniDSP 2x4 HD** (or similar) for crossover adjustments.
- **Cables**: USB cable for mic, RCA cables for subwoofer connection.

## Step 2: Set Up Measurement System

1. Place the measurement microphone at the main listening position (ear height, pointing up).
2. Connect the microphone to the laptop via USB.
3. In REW, go to *Preferences* > *Soundcard* and select the UMIK-1 as input. Load the calibration file (provided with the mic).
4. Set output to your AVR's subwoofer pre-out (or directly to sub if using MiniDSP).

## Step 3: Measure Room Response Without Crossover

1. In REW, click *Measure* > *Start Measuring*.
2. Set sweep parameters: frequency range 10-200 Hz, length 256k.
3. Play the sweep and capture the response. This gives the raw in-room bass response.
4. Save the measurement as "Raw Sub Response".

## Step 4: Analyze Room Modes and Peaks

1. In the REW graph, identify major peaks and nulls (room modes).
2. Note the frequency and amplitude of the strongest peak (e.g., 40 Hz +12 dB).
3. Use REW's *Room Simulation* tool to estimate modal frequencies if needed.

## Step 5: Determine Optimal Crossover Frequency

1. Set a target curve: typically flat from 20-80 Hz, then a gentle roll-off (e.g., -3 dB at 120 Hz).
2. In REW, use *EQ Filters* to find the best crossover point that minimizes overlap with mains.
3. Common starting point: 80 Hz (THX standard). Adjust based on measurement:
   - If a strong peak exists near crossover, lower crossover to avoid exciting that mode.
   - If mains have limited low-end, raise crossover to 100-120 Hz.
4. Save the target crossover frequency.

## Step 6: Configure MiniDSP Crossover

1. Connect MiniDSP to laptop via USB and open the MiniDSP plugin.
2. Set input routing: subwoofer channel to output 1 (or both if dual subs).
3. Apply a low-pass filter at the chosen crossover frequency (e.g., 80 Hz, Linkwitz-Riley 24 dB/oct).
4. Optionally add a high-pass filter for mains if using MiniDSP for mains too.

## Step 7: Apply EQ to Flatten Response

1. In REW, generate EQ filters for the subwoofer channel (PEQ) to correct major peaks.
2. Limit to 3-5 filters to avoid overcorrection.
3. Enter these filters into MiniDSP's PEQ section.
4. Re-measure and verify the response is within ±3 dB from 20-120 Hz.

## Step 8: Fine-Tune Phase and Delay

1. Use REW's *Acoustic Timing Reference* to measure phase alignment between sub and mains.
2. Adjust subwoofer delay (in MiniDSP or AVR) to achieve coherent summation at crossover.
3. Aim for a smooth transition with no dip or peak at crossover.

## Step 9: Final Verification

1. Play test tones or music with bass content.
2. Listen for localization (sub should not be heard as separate).
3. Use REW's *Waterfall* plot to check for excessive decay (ringing).
4. Save the final MiniDSP configuration file for future reference.

## Step 10: Document Settings

- Record crossover frequency, EQ filters, delay, and measurement graphs in the client's file.
- Provide a quick reference card for the homeowner.

## Tips for Success

- Always use a calibrated mic for accurate measurements.
- Take multiple measurements at different listening positions if multiple seats.
- For dual subs, measure each separately and then together to optimize placement and crossover.
- Use REW's *Alignment Tool* for multi-sub setups.
